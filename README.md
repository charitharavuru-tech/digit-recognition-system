# Handwritten Digit Recognition System

This project implements a Convolutional Neural Network (CNN) to recognize handwritten digits using the MNIST dataset.

## Technologies Used
- Python
- TensorFlow
- NumPy
- Streamlit
- Scikit-learn

## Features
- Image preprocessing and normalization
- CNN model training and evaluation
- Real-time digit prediction using a web interface
- High accuracy digit classification

## Dataset
MNIST dataset containing 70,000 handwritten digit images (0–9).

## Project Workflow
1. Data preprocessing and normalization
2. Model training using CNN
3. Model evaluation
4. Real-time digit prediction using Streamlit interface

## Purpose
This project demonstrates machine learning model development, data preprocessing, and deployment of an interactive prediction system.
